# Australian Road Crash Injury Visualisation

## Project Overview
This project explores hospitalised road crash injuries in Australia from 2011–2021 using interactive D3 visualisations.

## Features Planned
- Interactive charts
- Tooltips
- Filters
- Trend analysis
- Geographic comparisons

## Technologies
- D3.js
- HTML
- CSS
- JavaScript
- KNIME
